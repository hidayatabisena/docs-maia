## Mengubah Isi Karakter

Jika ingin melakukan perubahan konten atau edit karakter, caranya dengan klik tombol Character di kanan atas layar, lalu kita bisa melakukan Edit, Ekspor karakter dalam file berformat `json` ke komputer kita, atau menghapus karakter.

![Edit Character](https://res.cloudinary.com/moyadev/image/upload/v1732867624/maia/step7-changeCharacter_myf2ei.png)

1. Tombol untuk mengubah konten dari karakter
2. Tombol untuk ekspor karakter sebagai file json untuk disimpan di lokal komputer kita
3. Tombol untuk menghapus karakter

> Kita juga bisa import dari karakter yang sudah ada selama format filenya `json`
