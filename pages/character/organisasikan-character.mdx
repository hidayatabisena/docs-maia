# Buat Folder Character

Organisasikan karakter-karakter agar lebih rapi dan terstruktur kedalam folder-folder. Caranya dengan klik tombol folder berikut:

![Folder Character](https://res.cloudinary.com/moyadev/image/upload/v1732870160/maia/step10-folder_izdbjp.png)

## Memberi nama folder
Akan muncul popup untuk memberi nama folder sesuai yang diinginkan, lalu klik tombol `Create`

![Create Folder](https://res.cloudinary.com/moyadev/image/upload/v1732870412/maia/step11-nameFolder_fv0ugw.png)

## Drag karakter kedalam folder
Selanjutnya kita tinggal drag karakter-karakter yang ingin kita tambahkan kedalam folder yang telah dibuat:

![Drag folder](https://res.cloudinary.com/moyadev/image/upload/v1732870863/maia/step12-folderOrganize_ihrdbd.png)

Folder bisa diubah dan bisa juga dihapus. Namun **hati-hati** ketika sebuah folder dihapus, maka karakter didalamnya ikut terhapus. Namun jangan khawatir karena Anda bisa import karakternya lagi atau membuat karakter dari awal.
