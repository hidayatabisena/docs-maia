# Character Marketplace

Ada banyak karakter yang tersedia di fitur `Character Marketplace` yang bisa kita import dan gunakan di AI kita. Cara mengaksesnya melalui tombol karakter di kanan atas, lalu akan muncul fitur marketplace seperti ini:

![Character Marketplace](https://res.cloudinary.com/moyadev/image/upload/v1732869466/maia/step8-marketplace_emgklv.png)

akan muncul jendela popup seperti ini:

![Popup Window Character](https://res.cloudinary.com/moyadev/image/upload/v1732869833/maia/step9-popupWindow_bx7him.png)

1. Kita bisa search karakter dengan keyword tertentu
2. Setelah menemukan karakter yang dibutuhkan, klik tombol import
3. Scroll kebawah untuk melihat beragam karakter yang tersedia
