# Ekspor Data

![Data Export](https://framerusercontent.com/images/yUGdAWkQcArms4YfE2SI2hO5z5Q.png?scale-down-to=1024)

Anda dapat mengekspor data tabel sebagai CSV langsung dari antarmuka obrolan, tanpa perlu menyalin dan menempel tabel ke Excel. Hal ini membuat bekerja dengan data tabel menjadi lebih mudah dan nyaman.
