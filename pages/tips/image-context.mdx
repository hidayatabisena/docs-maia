# Chat dengan konteks berupa Image / gambar

Selain dengan dokumen, kita bisa menjadikan gambar sebagai konteks percakapan. Caranya sebagai berikut:

![Chat with Image](https://res.cloudinary.com/moyadev/image/upload/v1733058946/maia/tips5-chatImages_wiqiiq.png)

Tersedia tiga pilihan sumber gambar;
1. Dari galeri foto
2. Dari url
3. Dari kamera depan 

Kita contohkan dari foto galeri:

![Upload Image](https://res.cloudinary.com/moyadev/image/upload/v1733059378/maia/tips5-uploadImages_porqew.png)

Beberapa AI model yang support Vision (bisa menjadikan image sebagai konteks percakapan) diantaranya:
- OpenAI
- Antropic
- Google Gemini

![Ready to chat](https://res.cloudinary.com/moyadev/image/upload/v1733059553/maia/tips5-imageContext_rcfje0.png)

Lalu setelah itu kita coba dengan prompt berikut:

`Deskripsikan gambar ini`

Hasilnya kurang lebih seperti ini:

![Results chat with image](https://res.cloudinary.com/moyadev/image/upload/v1733059746/maia/tips5-results_r6zott.png)
