# Membagikan Chat ke Publik

Hasil dari chat dengan AI yang telah dilakukan bisa dibagikan kepada publik dengan fitur Share chat

![Share chat](https://res.cloudinary.com/moyadev/image/upload/v1732812325/maia/shareChat_lyfeml.png)

## Buat Link Chat

Klik tombol `Create Chat` untuk membuat link chat yang bisa dibagikan secara instan

![Create Link](https://res.cloudinary.com/moyadev/image/upload/v1732812462/maia/createLink_s8fsca.png)

## Salin Link Chat

Klik tombol `Copy Link` untuk menyalin link chat agar bisa diakses secara publik

![Copy Link](https://res.cloudinary.com/moyadev/image/upload/v1732812688/maia/copyLink_gi4hyb.png)
