# Dall-E 

Plugin Dall-E memungkinkan user untuk generate gambar AI langsung dari dalam thread chat tanpa harus mengganti model AI nya. Contohnya sebagai berikut: 

![DallE](https://res.cloudinary.com/moyadev/image/upload/v1733067155/maia/dall-e_lmsphp.png)

Namun perlu diingat bahwa gambar hasil generate dari plugin DallE akan expired dalam waktu 1 jam. 

Disarankan untuk menyimpan gambar segara di komputer Anda. 

Atau untuk history chat yang lebih lama, bisa memilih menggunakan model AI DallE nya langsung tanpa lewat plugin.
