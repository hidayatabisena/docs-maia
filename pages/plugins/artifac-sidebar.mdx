# Artifac Sidebar

Menampilkan preview code di Sidebar menu. Perbedaan antara Artifac Sidebar dan Artifac Render terletak pada posisi hasil render AI nya. 

Sesuai nama, Sidebar akan memunculkan preview render code di sisi kanan hasil chat, sedangkan Artifac Render akan menampilkan preview render code secara inline dengan hasil dari respon AI.
